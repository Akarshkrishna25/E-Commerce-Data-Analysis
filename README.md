# E-Commerce-Data-Analysis
This project presents a comprehensive exploratory, analytical, and transformation-based analysis of the Online Retail Dataset, a real-world transactional dataset representing e-commerce sales.
The objective is to clean, transform, analyze, and extract meaningful business insights using Python, Pandas, NumPy, and Excel-style reporting techniques.
